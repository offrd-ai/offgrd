Works directly with Hudl. In Hudl Classic → Reports / Breakdown, hit Export to Excel — upload the .xlsx directly (or save as .csv), then upload or paste it below. The importer auto-recognizes Hudl column names (DN, DIST, OFF FORM, OFF PERS, DEF FRONT, COVERAGE, BLITZ, YARD LN, ODK…). One row per snap; header row required.

Only coverage is required. Everything else is optional and auto-mapped:

  • Coverage auto-normalizes: Cov 3, C3, 3 Buzz, Quarters, 2 Man → Cover 3 / Cover 4 / 2-man, etc.
  • Pressure is inferred when a BLITZ/stunt column has a value (or use an explicit pressure 1/0).
  • Field zone reads from a zone column or is derived from YARD LN; an ODK column auto-filters to defensive snaps only.
  • After loading, the box below lists exactly which columns it mapped — check it once for a new Hudl export.
